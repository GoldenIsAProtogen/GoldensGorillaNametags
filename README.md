**GorillaNametags** is a mod that displays the players name, as well as FPS & mods (only works with mods that have custom properties)<br>
<br>
<br>
## Colors<br>
White = Dev9998, E14O<br>
White Gray = Zloth<br>
Gray = Outdated<br>
Blue = Legal<br>
Yellow = HanSolo1000Falcon<br>
Red = Illegal<br>
